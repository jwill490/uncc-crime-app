On a Mac with Xcode 7.1 installedUnzip project folder.Open itcs4102test.xcodeproj in xcode
On upper-section of the application, beside itcs4102test, select iPhone 6s Plus
Make sure “Allow Location Simulation” is checked in Product -> Scheme -> Edit Scheme

Set the Default Location to derp2 in Product -> Scheme -> Edit Scheme -> Default Location -> derp2
Click the Play button in the upper left corner to run the applicationIf the user’s location is not shown, the application may need to be restarted.
